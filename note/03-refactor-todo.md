  - 프레젠터 컴포넌트 : 화면 표현

## 개선하면서 기억해야할 것

- 안티 패턴 : 안좋은 패턴
  - 객체가 참조되서 메소드로 넘어올 때, 해당 참조된 객체의 프로퍼티를 바꾸는 것은 결합도를 높임
  - 따라서 한 컴포넌트 내의 데이터를 조작하는 것으로 (다른 컴포넌트 데이터 바꾸기 X) 리팩토링
